# SAPPhire

SAP 전문 인력과 기업을 연결, AI 기반 평가 및 매칭을 제공하는 채용 플랫폼

## Features

- 개인회원/기업회원/관리자 역할 기반 서비스 제공
- 일반 로그인 및 Google OAuth 로그인
- 전자 개인정보동의서 동의 및 약관 관리
- 이력서 작성, 수정, 삭제 및 공개 여부 설정
- 채용 공고 목록/상세 조회
- 기업 공고 등록, 수정, 삭제 및 임시저장
- 공고 지원 및 지원 내역 관리
- 기업의 공고별 지원자 관리
- 공개 이력서 열람 및 열람 기록 저장
- 포지션 제안 기능
- 공고 북마크 기능
- 프로필 이미지, 회사 로고, 이력서, 채용공고 첨부파일 업로드
- AI 기반 프로필 평가 결과 및 공고 매칭 점수 제공
- 관리자 회원, 공고, 지원 내역 관리
